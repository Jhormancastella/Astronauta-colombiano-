# Música de fondo

Coloca aquí los archivos de música en formato `.ogg` (recomendado) o `.mp3`.

Archivos esperados:
- `menu.ogg`      → música del menú principal
- `gameplay.ogg`  → música durante el juego

Recursos gratuitos recomendados:
- https://opengameart.org/content/space-music
- https://freemusicarchive.org
- https://incompetech.com (Kevin MacLeod, licencia CC)
