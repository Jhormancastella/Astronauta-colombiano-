# Efectos de sonido (SFX)

Coloca aquí los archivos de audio en formato `.ogg` (recomendado) o `.mp3`.

Archivos esperados:
- `shoot.ogg`      → sonido al disparar
- `explosion.ogg`  → explosión de fragmento
- `pickup.ogg`     → recoger item
- `hit.ogg`        → recibir daño
- `gameover.ogg`   → muerte del jugador

Recursos gratuitos recomendados:
- https://freesound.org
- https://opengameart.org
- https://sfxr.me (generador online de SFX retro)
